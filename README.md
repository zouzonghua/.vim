# vim
My Vim setttings

## 安装

### 一、手动安装

```sh
  cd ~
  git clone git@github.com:zouzonghua/.vim.git .vim
  ln -s .vim/vimrc .vimrc
```

打开 Vim，并执行 `:PlugInstall` 安装插件。

### 二、脚本安装

```
sh <(curl -L https://github.com/zouzonghua/vim/raw/main/utils/install.sh)
```
